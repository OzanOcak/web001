## Deployment of Next.JS App into Vercel

```console
npm i -g vercel

vercel
vercel --prod

```

Setting>Environment Variables

.env

```console
NEXT_PUBPLIC_SAMPLE=
NEXT_PUBPLIC_PASS=
```

or

```console
vercel env add NEXT_PUBPLIC_SAMPLE
```
